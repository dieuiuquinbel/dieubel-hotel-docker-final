`public/home/` chứa phần tách riêng cho trang chủ khách hàng.

- `trangChuData.js`: dữ liệu tĩnh của hero, điểm đến, đánh giá, bộ sưu tập.
- `trangChuHelpers.js`: helper gắn tham số tìm kiếm.
- `HomeVoucherCard.jsx`: khối hiển thị voucher.
- `HomeReviewPopupCard.jsx`: popup review chạy tự động.
- `HomeDialogs.jsx`: popup mời đăng nhập và popup kho voucher.
- `HomeSections.jsx`: các section lớn của trang chủ.
