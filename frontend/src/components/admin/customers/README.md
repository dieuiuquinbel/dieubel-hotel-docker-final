# Cấu trúc `components/admin/customers`

Nhóm component này phục vụ riêng cho màn hình `AdminCustomers.jsx`.

## Chức năng từng file

- `CustomerCreateForm.jsx`  
  Form tạo tài khoản khách trực tiếp từ admin.

- `CustomerList.jsx`  
  Danh sách khách hàng ở cột trái.

- `CustomerDetailPanel.jsx`  
  Panel chi tiết khách đang chọn ở cột phải.

- `customerStatus.js`  
  Hàm ánh xạ trạng thái khách hàng sang nhãn hiển thị và class màu.
