`bookings/` chứa các phần giao diện riêng cho màn quản lý đặt phòng.

- `bookingConstants.js`: nhãn tab và màu badge.
- `bookingHelpers.js`: helper lọc, đếm, nhãn nghiệp vụ.
- `BookingShared.jsx`: các khối UI dùng chung.
- `BookingQueueItem.jsx`: một đơn trong hàng đợi công việc.
- `BookingDetail.jsx`: panel chi tiết và thao tác quản lý.
